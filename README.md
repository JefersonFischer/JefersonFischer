### Sobre mim

Desenvolvedor Sênior de Software com mais de 15 anos de experiência, sendo os últimos 6 anos dedicados exclusivamente ao desenvolvimento backend em Java e Kotlin com arquitetura de microsserviços em ambientes de alta escala. Atualmente atuo como Desenvolvedor Especialista no **Grupo Boticário**, um dos maiores grupos de beleza e cosméticos do Brasil, onde contribuo com o design, desenvolvimento e evolução de APIs REST e microsserviços críticos para o negócio.

Possuo experiência consolidada com **Apache Kafka** para processamento de eventos distribuídos, **Spring Boot / Spring Cloud** para construção de serviços resilientes e observáveis, e **AWS** para deploy e operação de aplicações em nuvem. Sigo práticas de **TDD**, **SOLID** e **Clean Code** como padrão de trabalho, com foco em qualidade, manutenibilidade e desempenho.

Antes de me especializar em backend, atuei com automação industrial (C/C++, CLPs, IHMs), o que me proporcionou uma base sólida em pensamento sistêmico, integração de sistemas e resolução de problemas complexos em contextos de missão crítica.

- Proficiência em mensageria distribuída com Apache Kafka (idempotência, dead letters, correlation ids, batches)

- Comunicação eficaz em inglês para atuação em times globais
- Transição bem-sucedida de engenharia de automação para desenvolvimento de software — visão sistêmica aplicada ao backend
